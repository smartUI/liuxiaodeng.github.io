# 无宽度导航

无宽度导航，让导航更具有自适应，不再使用繁琐的计算，解放你的个位数加减计算能力。

导航一 导航二 导航三

通常看到上面的布局，都会这么做：

<template>
  <div class="demo1">
    <a class="nav">导航一</a>
    <a class="nav">导航二</a>
    <a class="nav">导航三</a>
  </div>
</template>
<style>
.demo1 {
  width: 200px;
  border-radius: 3px;
  background-color: #3eaf7c;
  
}
.demo1 .nav {
  display: block;
  width: 100%;
  border-bottom: 1px solid #0b7b48;
  color: #fff;
}
</style>

因为增加 width: 100% 破坏了元素的流动性，所以当我们增加一些边距时，会超出意外的效果。

导航一 导航二 导航三

.demo1 .nav {
  display: block;
  padding: 10px;
  width: 100%;
  border-bottom: 1px solid #0b7b48;
  color: #fff;
}

而我们去掉宽度，保持其流动性即可。

导航一 导航二 导航三

.demo1 .nav {
  display: block;
  padding: 10px;
  border-bottom: 1px solid #0b7b48;
  color: #fff;
}
