# 接受自己的平庸，用超视角看自己

有一位学生，在完成老师布置的论文作业时，信心满满的呈交给老师，因为自己付出了很大的努力，查阅了很多的资料。

过了几天，评论结果出来时自己大失所望竟然只是一个 C ，于是自己非常气氛的去找老师理论，我觉得我应该得 B，为什么你只给我一个 C。

结果老师拿过来直接把 C 划掉，然后写了一个大大的 B，然后很淡定的把论文递给他，在递过来的途中老师突然停住，然后问了我一句，你确定你不想要个 A ？

这个结果令他非常的差异，他认为老师会和他理论一番，因为自己也准备了很多应对的话语，结果老师非但没有反驳自己，而是非常豪爽的就这么给修改了，胜利来的竟然如此轻松。

显然，他面对这个情况时非常的懵逼不知所措，但后面更可怕的远不止于此。

过了一会，老师又说，这篇文章就是 C 的水平，不管我给什么它都是 C，你可以跟我说你以后想要什么评价，这样以后我也不需要去阅读你的论文了。

甚至，你也可以告诉我这门课你想要什么评价，这样你以后连课都不用上了。

事后，这位学生又恳求老师把分数修改了回来，因为它意识到自己的问题所在。

平庸没有什么不好，它比 D 要优秀，对比 A，B 还有很大的进步空间。

平庸没有什么不好，接受自己的平庸，会让我们变得更加谦虚。

任何事情都是相对的，大小也是这样，不管你在任何方面如何的优秀，都会有人比你更优秀。

百川汇入大河，大河觉得自己很牛逼，可是但它见到大海时，感受到了自己的渺小。

接受平庸，可以让我们向更好的人学习，拒绝平庸，只会阻碍我们自己。
