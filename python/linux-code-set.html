# Linux 常用命令集合

1. sudo：获取管理权限。
2. pwd：查看文件路径
3. clear：清屏
4. ./：当前路径
5. ../：上一层文件
6. cd -：传送至上次目录
7. cd ～：传送至 home 目录
8. cd /：传送至根目录
9. touch test.py：创建文件 
10. touch .test.py：//创建隐藏文件 .可用来隐藏文件
11. mkdir test：创建文件夹 
12. ls：显示文件夹下的文件
13. ls -a：显示全部文件
14. ls -l：以列表的方式显示文件的详细信息
15. ls -l -h：显示文件大小
16. ls -l -a -h：选项无选后顺序
17. ls -lah：ls -l -a -h的简写
18. ls --help：查看帮助文档
19. ls | more：内容分屏查看
20. tab：自动补全名称
21. 两次tab：可以补全全部文件名称
22. cat test.py：查看文件内容
23. cat file1 file2：查看多个文件
24. cat name1 name2 > xx.txt：利用重定向合并文件
25. more test.txt：查看文件内容，一般用于比较大文件，分屏显示
26. history：查看你用过的历史命名
27. !+历史序号：查看对应的执行命令
28. rm test.py：删除文件
29. rm *.py: 删除所有 .py 文件，\* 为通配符，匹配所有
30. rm -r *：递归删除文件下的所有文件
31. rm -rf：递归强制删除
32. a?.txt：以a开头的两位的名称，？为通配符，匹配一位
33. a[bcd].txt：以a开头，bcd为结尾的文件，[]为通配符，选择某一位
34. mkdir a/b/c/d/e -p：直接创建文件
35. rmdir test: 删除文件夹，非空文件
36. rm test -r：可删除文件文件夹，-r 递归删除
37. mv oldname newname：重命名
38. mv file dir：把文件移动到文件夹下
39. cp file dir：复制文件到对应文件夹下
40. cp dir dir -r：复制文件夹到文件夹下。
41. grep "abc" test.txt：在test.txt 中找出包含 abc 的内容
42. grep "\^abc" test.txt：在 test.txt 中找出 以 abc 开始的内容
43. grep "abc\$" test.txt：找出以 abc 结束的内容
44. >：重定向，把内容放到文件中。
45. ls > test.txt：把ls输出的内容放到test.txt中。
46. >>：追加到文件中，没有就新建
47. find：在指定目录下查找文件
48. find / -name "test"：寻找根目录下名字为 test 的文件。
49. tar -cvf test.tar *.py：打包文件
50. tar -xvf test.tar：解包文件
51. tar -zcvf test.tar.gz：压缩文件
52. tar -zxvf test.tar.gz：解压文件
53. tar -zxvf test.tar.gz -C dir/：解压到指定目录
54. cal：查看日历
55. cal -y 2018：2018的12个月
56. date：查看当前时间
57. date "+%Y-%m-%d"：指定输出格式
58. ps：进程查看
59. ps -aux：全部进程
60. top：查看进程的资源占用状况
61. htop：查看进程的资源占用状况
62. kill pid：杀死进程
63. kill -9 pid：强制杀死
64. shutdown -h now：关机
65. reboot：重启
66. df -h：硬盘被占用了多少空间
67. ifconfig：查看IP地址信息
68. ping：查看通讯是否正常
69. ifconfig | grep 192：查看包含 192 的内容。
70. cat /etc/passwd：查看所有用户
71. whoami：查看是谁
72. exit：推出账户
73. ssh user@192.168.1.1：远程连接电脑
74. who：查看终端登录
75. useradd username -m：添加账户
76. passwd username：设置账户密码
77. su username：切换账户
78. su - username：切换账户和目录
79. userdel username：删除用户
80. userdel username -r：连删除目录
81. groupadd groupdel：新建/删除组
82. chgrp group filename：改变文件或目录所属的组
83. chown username/group file：将指定文件的拥有者改为指定的用户或组
